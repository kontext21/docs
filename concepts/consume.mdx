## Usage / Consuming the data 

After processing or contextualization the data is ready to be consumed and be used for your application.

support different ways to consume the data:

### Pull
- Query database (local sqlite or cloud)
- Interprocess commmunication 
- API 
- File system

### Push
- Interprocess commmunication 
- Streaming (Websocket)
- APIs Webhook - we can post the data to your custom API 
- File system

## Frequency
- Real-time
- Batch (re-curring)
- on demand (pull)
- ad-hoc (push) 
- Scheduled/cron

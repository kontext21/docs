Kontext21 is the AI infrastructure layer for user context. 

Kontext21 SDK makes it easy to personalize software by capturing & retrieving end-user context from sources like their screen, browser and many more. 

<AccordionGroup>
  <Accordion title="Questions you can answer with Kontext21" defaultOpen={false} icon="question">
  Kontext21 SDK let’s developers answer questions like:

    * “What is my user currently doing?”

    * “Which contacts are important to her?”

    * “What is his most likely next action?”
  </Accordion>

  <Accordion title="Example - AI Task Tracking" defaultOpen={false} icon="circle-info">
    * Todo Inc. offers a todo & time tracking web app

    * Todo's users are frustrated to manually start & stop tasks <br/> &lt;&gt; Instead they expect AI and automation!

    * With Kontext21 Todo can build automated task tracking easily
  </Accordion>
</AccordionGroup>

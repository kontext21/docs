Kontext21 is the AI infrastructure layer for user context.

K21 SDK makes it easy to capture & retrieve end-user context from sources like their screen, browser and many more. The contextualized, structured data can then be used to personalize software and AI apps.

<AccordionGroup>
  <Accordion title="Example - AI Task & Time Tracking" defaultOpen={false} icon="circle-info">
    * Todo Inc. offers a todo & time tracking web app

    * Todo's users are frustrated to manually start & stop tasks <br /> \<> Instead they expect AI and automation!

    * With Kontext21 Todo can build automated task tracking easily
  </Accordion>

  <Accordion title="Questions you can answer with K21 SDK" defaultOpen={false} icon="question">
    K21 SDK let’s developers answer questions like:

    * “What is my user currently doing?”

    * “Which contacts are important to her?”

    * “What is his most likely next action?”

</AccordionGroup>

## Install

```sh
npm install @kontext21/k21
```

## Use

```javascript
const { K21 } = require('@kontext21/k21');

const k21 = new K21();
k21.setCapturer()
k21.setProcessor()
const result = await k21.run()
// defaults to screen capture 
// and extract text with local OCR

console.log(result)
```

## Overview

The K21 Node.js SDK comes with a comprehensive sample project that demonstrates various features and implementation patterns. This sample repository serves as a practical reference for integrating K21 into your applications. The official sample repository is available at [kontext21/k21-node-sample](https://github.com/kontext21/k21-node-sample)

## Getting Started with Examples

1. Clone the repository:
```bash
git clone https://github.com/kontext21/k21-node-sample.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the examples:
```bash
npm run start
```

## Available Examples

The sample repository includes implementations of:

- Basic screen capture
- OCR processing
- Vision model integration
- File-based capture
- Custom processor configuration

Each example is documented with comments and includes configuration options for different use cases.

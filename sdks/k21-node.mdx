Use this SDK to capture and process user screen from your node application.

```bash
npm install @kontext21/k21
```

Also see the [Github repo](https://github.com/kontext21/k21-node) .

# Functions
## ping

Super simple function to check if the SDK is working.

```js
const k21 = require('@kontext21/k21');

k21.ping();
```

## process_image

```js
const k21 = require('@kontext21/k21');

k21.process_image("../image.png");
```

## process_video

```js
const k21 = require('@kontext21/k21');

k21.process_video("../video.mp4");
```
## record_screen_images	

```js
const k21 = require('@kontext21/k21');

k21.record_screen_images();
```
